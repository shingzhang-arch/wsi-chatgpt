Known Issues & Fixes

Developer checklist for addressing current production issues.

  • 1
    Blurred card edges
    Replace box-shadow blur with crisp border: 1px solid #E0E0E0. Use border-radius: 12px with overflow: hidden. Serve @2x images for retina. Optionally add image-rendering: -webkit-optimize-contrast.
  • 2
    Cards not mobile-friendly
    Switch from fixed-width to fluid widths per breakpoint. Use calc(100vw - 32px) or media-query values from Breakpoint Reference. Enable scroll-snap for clean swipe.
  • 3
    Video wrong size on mobile
    Use aspect-ratio: 16/9 container with width: 100%. Never set fixed pixel height on iframe. Add max-width: 100% to prevent overflow.
  • 4
    Unwanted scrollbar
    Carousel: scrollbar-width: none and ::-webkit-scrollbar { display: none }. Video: overflow: hidden with proper aspect-ratio container.
